# 数据下载指引

## 所需数据集
**Crimes in Chicago**（芝加哥犯罪数据集）
- 数据集页面：https://www.kaggle.com/datasets/currie32/crimes-in-chicago
- 文件：Crimes_-_2001_to_Present.csv（约 700MB）

## 下载步骤
1. 打开上面的链接（如未登录 Kaggle 请先注册/登录）
2. 点击页面右侧的 "Download" 按钮
3. 将下载的 Crimes_-_2001_to_Present.csv 放入本目录（data/）
4. 文件名必须保持不变，否则 Notebook 中的代码无法找到文件

## 如果 Kaggle 访问慢
可以尝试：
- 使用 Kaggle 国内镜像（搜索 "Kaggle 镜像"）
- 或使用其他城市开放犯罪数据替换分析代码中的数据加载部分
